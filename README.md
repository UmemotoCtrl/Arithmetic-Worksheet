# 算数プリント瞬間生成

本ツールは営利目的・商用利用に関わらず無料で使用可能です。利用は自己責任でお願いします。
平文テキストで作成した問題を簡易的にPDF組版します。基本的に1問1行で記載できる問題をきれいにレイアウトするようになっています (筆算不可、複数行問題も一応レイアウト可能)。

## 使い方

1. ローカルファイルの読み込み、またはテキストエリアから問題作成。サンプルファイルをいくつかダウンロード可能。時短のためにAI Chatや表計算ソフトを活用しましょう。
2. ラジオボタンで段組みを選択。
3. Previewボタンを押し内容を簡易的に確認。
4. Printボタンで印刷画面から印刷やPDF出力。印刷時に自動で倍率が調整される場合があります。"規定"または"100%"などを選択してください。

## 問題の作り方

* 1行1問として扱われます。縦には1行問題が15問入るように調整してあります。
* 特別な記号
    * %1sで1文字、%3sで3文字分スペースを空けます。
    * \pagebreakで改ページします。
    * HTMLタグは有効で装飾も可能です。&lt;br&gt;で問題中での改行、&lt;b&gt;&lt;/b&gt;で太字等。
* 1段組みは左寄せ、2段組みは均等割り付けします。均等割り付けの場合、半角スペースの幅は自動調整されます。連続した半角スペースは1つとして扱われます。全角スペースは連続していてもそれぞれ有効。
