# 算数プリント瞬間生成

本ツールは営利目的・商用利用に関わらず無料で使用可能です。利用は自己責任でお願いします。
平文テキストで作成した問題を簡易的にPDF組版します。1問1行で記載できる問題に対応しています (筆算不可)。


## 使い方

1. ローカルファイルの読み込み、またはテキストエリアから問題作成。サンプルファイルをいくつかダウンロード可能。
2. ラジオボタンで段組みを選択。
3. Generateボタンを押し内容を簡易的に確認。
4. Printボタンで印刷画面から印刷やPDF出力。

## 問題の作り方

* 1行1問として扱われます。
* %1sで1文字、%3sで3文字分スペースを空けます。
* 1段組みは左寄せ、2段組みは均等割り付けします。均等割り付けの場合、半角スペースの幅は自動調整されます。連続した半角スペースは1つとして扱われます。全角スペースは連続していてもそれぞれ有効。
* HTMLタグは有効で装飾も可能です。
